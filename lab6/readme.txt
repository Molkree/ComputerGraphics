Формат файла для многограника:
N
m1 x1 y1 z1 x2 y2 z2 .. x_m1 y_m1 z_m1
...
mn x1 y1 z1 x2 y2 z2 .. x_mn y_mn z_mn
где N - количество граней, далее в N строках m_i - количество точек в грани, далее m_i троек

Формат файла для фигуры вращения:
N
x1 y1 z1 x2 y2 z2
M
x1 y1 z1 x2 y2 z2 ... xm ym zm
где N - число разбиений, далее ось (задана двумя точками)
M - число точек, далее m троек
